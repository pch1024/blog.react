# 自定义脚手架流程

定义项目基本信息

> npm init

## 使用的工具

* tool:webpack
* html:vue
* css:postcss
* js:ts
* ajax:axios

## Webpack

1.  要安装最新版本，如果你使用 webpack 4+ 版本，你还需要安装 CLI。

```shell
npm i webpack webpack-cli -D
```

2.模块热替换

```shell
npm i webpack-dev-server -D
```

2.  HTML

```shell
npm i -D html-webpack-plugin
```
